## Типы
### IApi
Методы взаимодействия с API:
- getProducts() - возвращает промис с массивом продуктов (IProduct[])
- getProductById(id: string) - получает продукт по id, возвращает промис с объектом IProduct
- postOrder(order: IOrderForm) - отправляет заказ, возвращает промис с объектом IOrderForm

### IBasket
Структура товаров корзины:
- productsInBasket - список товаров в корзине
- totalPrice - итоговая цена всех товаров в корзине

### IGlobalState
Структура состояния сайта:
- basket - объект типа IBasketModel, содержащий информацию о корзине
- order - объект типа IOrderFormModel, содержащий информацию о заказе
- catalog - объект типа ICatalogModel[], содержащий информацию о списке всех товаров на странице

### IOrderForm
Структура заказа:
- payment - строка, обозначающая вид оплаты
- email - строка, обозначающая почту
- phone - строка, обозначающая номер телефона
- address - строка, обозначающая адрес
- total - число, обозначающая итоговая цена заказа
- items - массив строк, содержащий ID товаров

### IProduct
Структура продукта:
- id - строка, обозначающая ID товара
- description - строка, обозначающая описание
- image - строка, обозначающая ссылку на изображение
- title - строка, обозначающая заголовок
- price - число, обозначающая цену товара
- category - строка, обозначающая категорию товара
- isSelected - булевое значение, показывающее, выбран товар или нет

## Модели
### IBasketModel
Методы работы с корзиной:
- basket — объект с текущими данными корзины
- add(id: string) — метод, добавляющий товар в корзину по ID
- changeAmount(id: string, amount: number) — изменяет количество товаров в корзине 
- remove(id: string) — удаляет товар из корзины по ID 
- clear() — очищает корзину

### IOrderFormModel
Методы работы с заказом:
- updateField(name: string, value: any) — обновляет значение поля по его имени
- clearField(name: string) — очищает конкретное поле формы
- clearAll() — сбрасывает все поля формы

### ICatalogModel
Методы работы с каталогом товаров:
- getProductById(id: string) - принимает id товара и возвращает промис продукта по id, делая проверку на null
- getAllProducts() - возвращает промис списка всех продуктов на странице
